### Visão Geral

`comfy-cli` é uma [ferramenta de linha de comando](https://github.com/Comfy-Org/comfy-cli) que facilita a instalação e o gerenciamento do Comfy...

### Instalar CLI

<InstallCli/>

### Instalar ComfyUI

<InstallComfyFromCli/>

### Executar ComfyUI

```bash
comfy launch
```

### Gerenciar Nós Personalizados

```bash
comfy node install <NODE_NAME>
```

Usamos o `cm-cli` para instalar nós personalizados. Veja a [documentação](https://github.com/ltdrdata/ComfyUI-Manager/blob/main/docs/en/cm-cli.md) para mais informações.

### Gerenciar Modelos

Baixar modelos com o `comfy-cli` é fácil. Basta executar:

```bash
comfy model download <url> models/checkpoints
```

### Análise de Dados

Nós rastreamos o uso da CLI para melhorar a experiência do usuário. Você pode desativar isso executando:

```bash
comfy tracking disable
```

Para reativar o rastreamento, execute:

```bash
comfy tracking enable
```
