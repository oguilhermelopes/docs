[Instale o Miniconda](https://docs.anaconda.com/free/miniconda/index.html#latest-miniconda-installer-links). Isso o ajudará a instalar as versões corretas do Python e outras bibliotecas necessárias para o ComfyUI.

Criar um ambiente com o Conda.

```
conda create -n comfyenv
conda activate comfyenv
```
