
**Utilização**:

```console
$ comfy node [OPÇÕES] COMANDO [ARGS]...
```

**Opções**:

* `--install-completion`: Instalar conclusão para o shell atual.
* `--show-completion`: Exibir conclusão para o shell atual, para copiá-la ou personalizar a instalação.
* `--help`: Mostra as opções disponiveis.

**Comandos**:

* `deps-in-workflow`
* `disable`
* `enable`
* `fix`
* `install`
* `install-deps`
* `reinstall`
* `restore-dependencies`
* `restore-snapshot`
* `save-snapshot`: Salvar um instantâneo do ComfyUI atual...
* `show`
* `simple-show`
* `uninstall`
* `update`

### `deps-in-workflow`

**Utilização**:

```console
$ deps-in-workflow [OPÇÕES]
```

**Opções**:

* `--workflow TEXT`: Arquivo de fluxo de trabalho (.json/.png)  [obrigatório]
* `--output TEXT`: Arquivo de fluxo de trabalho (.json/.png)  [obrigatório]
* `--channel TEXT`: Especifique o modo de operação
* `--mode TEXT`: [remote|local|cache]
* `--help`: Mostra as opções disponiveis.
