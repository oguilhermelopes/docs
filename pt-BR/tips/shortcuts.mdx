<Tabs>
<Tab title="Windows/Linux">
| Atalho                             | Explicação                                                                                                         |
|------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| Ctrl + Enter                       | Colocar o gráfico atual na fila para geração                                                                       |
| Ctrl + Shift + Enter               | Colocar o gráfico atual na fila como o primeiro para geração                                                       |
| Ctrl + Z / Ctrl + Y                | Desfazer/Refazer                                                                                                   |
| Ctrl + S                           | Salvar fluxo de trabalho                                                                                           |
| Ctrl + O                           | Carregar fluxo de trabalho                                                                                         |
| Ctrl + A                           | Selecionar todos os nós                                                                                            |
| Alt + C                            | Recolher/expandir nós selecionados                                                                                 |
| Ctrl + M                           | Desativar/ativar nós selecionados                                                                                  |
| Ctrl + B                           | Desviar nós selecionados (funciona como se o nó tivesse sido removido do gráfico e os fios fossem reconectados)    |
| Delete<br/>Backspace               | Excluir nós selecionados                                                                                           |
| Ctrl + Delete<br/>Ctrl + Backspace | Excluir o gráfico atual                                                                                            |
| Space                              | Mover a área de trabalho                                                                                           |
| Ctrl + Clique<br/>Shift + Clique   | Adicionar nó clicado à seleção                                                                                     |
| Ctrl + C/Ctrl + V                  | Copiar e colar nós selecionados (sem manter as conexões com os nós não selecionados)                               |
| Ctrl + C/Ctrl + Shift + V          | Copiar e colar nós selecionados (mantendo as conexões dos nós não selecionados aos nós colados)                    |
| Shift + Arrastar                   | Mover vários nós selecionados ao mesmo tempo                                                                       |
| Ctrl + D                           | Carregar gráfico padrão                                                                                            |
| Q                                  | Alternar visibilidade da fila                                                                                      |
| H                                  | Alternar visibilidade do histórico                                                                                 |
| R                                  | Atualizar gráfico                                                                                                  |
| Duplo clique<br/>(botão esq.)      | Busca rápida                                                                                                       |
</Tab>
